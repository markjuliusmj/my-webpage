WS1 ACTIVITY 06 Part 3 - Definition List

DEFINITION LIST

COMPUTER TERMS

CPU
Central Processing Unit, the brain of a computer that executes instructions.
RAM
Random Access Memory, temporary storage for data the computer is actively using.
Hard Drive
Long-term storage for files and programs.
Software
Instructions that tell a computer what to do, like programs and apps.
Hardware
The physical components of a computer, like the keyboard and monitor.
Internet
A global network of computers that allows communication and data sharing.
Website
A collection of web pages accessible on the internet.
Browser
A program used to access and view websites.
Algorithm
A set of steps or rules used to solve a problem or complete a task.
Firewall
Security software that protects a computer from unauthorized access.
